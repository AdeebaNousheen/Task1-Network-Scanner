# 🔐 Task 1 - Network Scanner

## 📌 Project Overview

This project is a **Python-based Network Scanner** developed as part of my **Cybersecurity Internship**.

The scanner uses the **Nmap** library to scan a target IP address, identify open ports, detect running services, and save the scan results into a JSON file.

---

## 🚀 Features

✅ Scan a target IP address

✅ Detect open ports

✅ Identify running services

✅ Export scan results to JSON format

✅ Simple and beginner-friendly Python project

---

## 🛠️ Technologies Used

- 🐍 Python
- 🔎 Nmap
- 📄 JSON

## ▶️ How to Run

### 1️⃣ Install Nmap

Download and install Nmap.

### 2️⃣ Install Python Library

```bash
pip install python-nmap
```

### 3️⃣ Run the Scanner

```bash
python network_scanner.py
```
